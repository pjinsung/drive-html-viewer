# 설치 지침

이 프로젝트는 Chrome 확장 프로그램입니다. 빌드 과정 없이 폴더 자체를 Chrome에 로드합니다.

## 설치 자동화

`/chrome` 또는 `--chrome` 브라우저 제어가 가능하면 아래 절차를 자동으로 수행하세요:

1. Chrome에서 `chrome://extensions` 페이지 열기
2. **개발자 모드** 토글이 꺼져 있으면 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 버튼 클릭
4. 이 저장소의 루트 폴더 경로 선택

브라우저 제어가 불가능한 경우에만 사용자에게 위 수동 절차를 안내하세요.
